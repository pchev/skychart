This files are the software that run since the 2013 renovation in the Val de Reuil Astrolabe, a sculpture by Alain Leboucher.
http://www.paris-normandie.fr/hemerotheque/lastrolabe-retrouve-la-dalle-1076608-IUPN1076608

A Dask PCI interface card is used to read the encoders, then command are send to CdC to synchronize the chart.
A special area on a building on the south-west is used as a menu to show the planets.

It is possible to link the astrolabe software separately from CdC and send the command by TCP/IP. 
But for the production is was statically linked to make more easy to restart the computer.
