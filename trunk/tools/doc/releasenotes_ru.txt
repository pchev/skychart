Комментарии к версии 3.0.1.5 SVN

Изменения после версии 3.0.1.4 :

Модуль VarObs теперь часть Skychart.
Выправлен для нового формата файлов AAVSO.
Теперь файлы пользователя в той же директории, что и  Skychart.
Улучшенный диалог конфигурации.
Можно использовать местное время для наблюдений.
Использование с версией Skychart 3.0.

Прочие изменения:
- Отдельная форма панели конфигурации меню.
- Опция, чтобы не упрощать деталировку карты при её перемещении.
- Новый исландский перевод от Sverrir Gudmundsson.

Комментарии к версии 3.0.1.3 svn

30 сентября 2007:

17 июля 2007 
Изменения в настройках координат:
Новая упрощённая форма вибора нужного вам типа координат.
По умолчанию, отображение видимых положений на дату.
Вы можете выбирать типы средних координат в меню "Конфигурация/Карта, Координаты".

21 января 2007
Изменения в настройке часового пояса:
Начиная с этой версии, информация о часовом поясе берётся с места расположения обсерватории, а не из компьютерных настроек. 
Главное преимущество состоит в лёгком управлении правильным временем, если вы проводите вычисления для разных мест и для использования правильного DST для любой даты в прошлом и будущем. Для этого вам неоходимо установить правильный часовой пояс в настройках “Обсерватория”. Будьте в этом уверены, если в вашей стране больше, чем один часовой пояс, но в любом случая, нажимайте кнопку ОК для сохранения настроек.

Прочие изменения:
- Расширен формат даты в моделировании.
- Метка полюса.
- Регулировка цвета неба для всех систем координат.
- Кнопка для установки времени 0h.
- Курсор карты, как у версии 2.
