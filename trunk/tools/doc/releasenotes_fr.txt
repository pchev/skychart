Version 3.1  Notes de la version

Le but de cette version est principalement de corriger tout les bugs bloquant pour une première version stable.
Mais il y a quand même quelques changements dans les fonctionnalités qui sont décrits ici:

Interface ASCOM intégrée, peut être utilisée avec Windows x64. 
Support pour INDI 0.6
Plus d'options de la ligne de commande pour permettre des simples scripts d'automatisation.
Ajout des symboles pour les objets du système solaire.
Modification des couleurs par défaut et du réglage des étoiles.
Permet de changer les images des étoiles en mode photographique.
Permet de ne pas remplir le dessin des nébuleuses en mode graphique.
Ajout de raccourcis clavier.
Un clic droit sur les boutons permet d'accéder directement au réglages correspondant.
Plus de serveur d'images DSS.
Mise à jour de la base de donnée SAC à la version 8.1.
Sur Mac OS X le répertoire utilisateur change de ~/.skychart vers ~/Library/Application Support/skychart
Beaucoup de correction de bug.

Le détail des changement est ici:
http://www.ap-i.net/mantis/changelog_page.php?project_id=1
