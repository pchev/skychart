Version 3.0.1.5 SVN  Notes de la version

Changements depuis la version 3.0.1.4:

Module pour les étoiles variables VarObs:
VarObs fait maintenant partie de Cartes du Ciel.
Modifié pour les nouveaux format de fichiers de l'AAVSO.
Les fichiers utilisateurs sont maintenant dans le même répertoire que Cartes du Ciel.
Amélioration du dialogue de configuration.
Possibilité d'utiliser l'heure locale pour les observations.
Marche avec Cartes du Ciel version 3.0.

Autres changement;
Séparations des options de configuration dans le menu principal.
Option pour ne pas simplifier la carte pendant le déplacement.
Traduction en Islandais par Sverrir Gudmundsson.
