Version 3.0.1.5 SVN  Notes de la version

Changement du 18 octobre:

Pour réduire le chevauchement des labels provenant de différentes sources sont écrit a différentes position autour de l'objet.
Il est maintenant possible d'activer les labels pour les étoiles, les variables et les doubles sans chevauchement.
Pour les nébuleuses le label SAC est écrit à droite et le label des autres catalogue à gauche.
Il y a une option dans Configuration/Affichage/Label pour afficher les labels de tous les catalogues d'étoiles.
+ correction de bug selon la roadmap: http://www.ap-i.net/mantis/roadmap_page.php

Changement du 21 septembre:

Ajout de l'aide contextuelle, appuyez sur F1.
Ajoute a nouveau la fenêtre principale de configuration.
Corrige le choix du port du télescope sous Linux.
Ajout de l'option de ligne de commande --config pour indiquer le fichier .ini à utiliser.
Ajout de l'option de ligne de commande --unique pour n'avoir qu'une seule instance du programme.

Changements depuis la version 3.0.1.4:

Module pour les étoiles variables VarObs:
VarObs fait maintenant partie de Cartes du Ciel.
Modifié pour les nouveaux format de fichiers de l'AAVSO.
Les fichiers utilisateurs sont maintenant dans le même répertoire que Cartes du Ciel.
Amélioration du dialogue de configuration.
Possibilité d'utiliser l'heure locale pour les observations.
Marche avec Cartes du Ciel version 3.0.

Autres changement;
Séparations des options de configuration dans le menu principal.
Option pour ne pas simplifier la carte pendant le déplacement.
Traduction en Islandais par Sverrir Gudmundsson.
