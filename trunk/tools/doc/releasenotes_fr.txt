Version 3.11 beta, Notes de la version

Nouvelles fonctions:

  * L'éditeur de barre d'outils vous permet de modifier la disposition des boutons. La fonction est disponible depuis la menu Configuration / Éditeur de barre d'outils.


Corrections:

  * N'enregistre plus l'emplacement des données du programme dans le fichier de configuration. Si vous avez besoin d'un emplacement non standard utilisez la nouvelle option de ligne de commande "--datadir=".
