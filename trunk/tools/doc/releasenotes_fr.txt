Version 3.11 beta, Notes de la version

Incompatibilités avec les versions beta inférieur à 3023:

  * Modification des boîtes à outils:
    Seul le script F1 est maintenant affecté par défaut. 
    Pour en charger d'autres voir http://www.ap-i.net/skychart/fr/documentation/toolbox
   
  * Si vous avez développé vos propres scripts:
    Le format de fichier des scripts est modifié par la version 3.11-3023. Vous devez enregistrer vos propres scripts à nouveau parce que le support de l'ancien format sera supprimé avant la sortie de la version stable.
    De plus, le code des scripts n'est plus sauvegardé dans skychart.ini. En cas de problème, vous pouvez utiliser la copie de sauvegarde avec l'extension .oldscript qui se fait la première fois que vous exécutez cette version.

Nouvelles fonctions:

  * Utilise INDIstarter pour gérer le serveur INDI.
  * Nouveau panneau de controle INDI.
  * Nouvelle option de performance depuis le menu Fichier/Carte et options par défaut...
  * Nouvelles boites à outils que vous pouvez activer avec les touches F1 à F8.
  * Fonctions de script et boites à outils personnalisées.
  * L'éditeur de barre d'outils vous permet de modifier la disposition des boutons. La fonction est disponible depuis la menu Configuration / Éditeur de barre d'outils.
  * Nouvelle représentation pour la Voie Lactée. Vous pouvez l'activer à partir du menu Configuration / Affichage / Lignes, en décochant "Voie lactée - Mode ligne".
  * Possibilité d'ajouter un réticule au centre de la carte. 
  * Option pour ajouter la ligne méridienne avec toute projection.
  * Ajout de la connexion Tcp/Ip à l'interface télescope LX200. 
  * Affiche le déplacement horaire des comètes et astéroïdes dans la fenêtre de détail.

Corrections:

  * N'enregistre plus l'emplacement des données du programme dans le fichier de configuration. Si vous avez besoin d'un emplacement non standard utilisez la nouvelle option de ligne de commande "--datadir=".
