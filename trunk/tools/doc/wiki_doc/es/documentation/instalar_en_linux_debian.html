Instalar en Linux Debian

El paquete para Debian está disponible para permitir la instalación automática utilizando apt-get

Primero, deben ser root y editar el archivo /etc/apt/sources.list para agregar la siguiente línea: 

deb http://www.ap-i.net/deb/ skychart/

Entonces actualicen la lista de paquetes, desde la línea de comando escriban : 

apt-get update

Instalen el software necesario previamente (pre-requisitos). Normalmente cualquier instalación Debian reciente con un server X está ok. Pueden tener que agregar los paquetes libgtk1.2, libgdk-pixbuf2, libjepg62 y libsqlite3-0 

apt-get install llibgtk1.2 libgdk-pixbuf2 ibjpeg62 libsqlite3-0

e instalen el software : 

apt-get install skychart

y si tienen mucho espacio: 

apt-get install skychart-pictures

Y eso es todo, salgan del shell de root y escriban skychart para ejecutar el programa.

Nota: cuando hagan un upgrade de una versión anterior a v3 beta 0.1.0 el upgrade no es automático porque cmabian las dependencias. En este caso, instalen el paquete de pre-requisitos como se describió anteriormente.
