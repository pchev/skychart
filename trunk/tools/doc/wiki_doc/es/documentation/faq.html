Instalación

Tengo CdC versión 2.76. ¿Que necesito para ejecutar la nueva versión 3?. ¿Que archivos tengo que descargar y tengo que borrar la 2.76 previamente?

Solo necesitas instalar la versión 3 beta para Windows en el mismo directorio que la versión 2.76. No intentes instalar en otro directorio y no borres nada, ya que puedes utilizar las dos versiones al mismo tiempo, esto es muy útil hasta tanto la versión V3.0 no esté finalizada. Ejecuta cdc.exe para la versión V3.0 o ciel.exe para la versión V2.76

¿Los catálogos asociados a CdC V2.76 son compatibles con CdC V3.00 beta?

Si, todos los catálogos pueden ser utilizados con la versión 3.0

¿Dónde instalo los catálogos extra en Linux, en que directorio se almacenan?

Si puedes obtener permiso de root es mejor colocarlos en /usr/share/apps/skychart/cat para juntarlos con los catálogos básicos. Pero puedes también instalarlos donde prefieras, ya que puedes especificar el camino en el menú de configuración de catálogos.

¿Puedo obtener Cartes du Ciel para Macintosh?

Recientemente compilé y ejecuté CdC sobre OSX 10.4/Intel. Probablemente también sea posible hacerlo con versiones PPC anteriores. Pero esto aún requiere de trabajo y no soy usuario de Mac, por favor, contáctame si posees alguna habilidad de programación y deseas ayudar con este port.

¿Que puedo hacer con MySQL?

MySQL se utilizó como la base de datos inicial hasta la versión alpha 7. Desde Junio de 2005 se reemplazó con la más simple SQLite. Aún puedes utilizar MySQL si lo prefieres, por ejemplo, para compartir la base de datos en una red local. La base de datos se utiliza para almacenar la siguiente información: elementos de asteroides y cometas, imágnes de objetos, ubicación en el mundo.

Utilización

¿CdC V3.0 para Linux soporta el seguimiento de telescopio como V2.76?

Si, CdC V3.0 para Linux utiliza INDI como interface con el telescopio. Cualquier modelo de telescopio soportado por INDI debería trabajar con CdC.

No hay un “seguir” y “ver” en el menú telescopio en V3.0

Esto se reemplazó por la característica “fijar en” (el botón con el ancla). Cuando el telescopio se conecta y no se selecciona un objeto este botón fija el telescopio.

¿Porque el cielo es azul?

Porque es de día o la Luna está alta. Puedes cambiarlo en el menú configuración, Mostrar, Color de Cielo.

¿Puedo ver cartas celestes con planetas antes de 3000 A.C?

Esto no es posible, ya que no disponemos de un método de cómputo confiable para antes de 3000 AC. Incluso la efeméride más larga de JPL DE406 no va más atrás.

¿Cómo puedo cambiar de Inglés a mi lenguaje?

Desde la versión beta 3.0.1.2 el lenguaje se detecta automáticamente cuando inicias el programa. El menú Configuración / Sistema / Lenguaje le permite seleccionar otro lenguaje.

¿CdC V3 soporta DDE?

No por el momento. El protocolo de comunicación standard para la versión 3 es TCP/IP por una cuestión de portabilidad. DDE se implementará probablemente en el futuro, probablmente como una interfaz.
