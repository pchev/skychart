Instalar binarios Windows

Instrucciones para instalar los binarios en Windows

La forma recomendada es instalar primero la versión de Windows 2.76 completa, disponible en http://www.stargazing.net/astropc

Después ejecutar el archivo instalador skychart_3.xxxxx.exe e instalarlo en el mismo directorio (Por ej. C:\Archivos de Programa\Ciel)

Se pueden utilizar indistintamente la versión 3 ejecutando cdc.exe o la versión 2.76 ejecutando ciel.exe

También puedes instalar sólo la versión 3, pero en este caso necesitarás instalar skychart_3.xxxxx.exe y luego extraer cdc_base_catalog.tar.gz al mismo directorio.

Si no posees un software que maneje archivos .tar.gz en Windows, puedes obtener 7-zip en: http://sourceforge.net/projects/sevenzip/
