Instal·lar en Mac OS X

Requeriment previ

  • Un Mac recent amb processador Intel, processadors PPC no estan suportats de moment.

* Mac OS X 10.4 o millor.

Instrucció per instal·lar els binaris en Mac OS X.

  • Descarregueu el fitxer d'instal·lació, i.e. skychart-3.0.1.4-i386-macosx.dmg
  • Obriu el fitxer per muntar el disc virtual.
  • La carpeta del disc oberta conté una carpeta “Cartes du Ciel”.
  • Copieu aquesta carpeta a la vostra carpeta d'Aplicacions.
  • Ara podeu desmuntar el disc virtual (Arrossegueu-lo a la paperera).

Començar el programa

  • Obriu la carpeta Aplicacions - Cartes du Ciel.
  • Obriu la icona “skychart”.

Limitacions

  • No funciona el control de telescopis.
  • La descàrrega d'imatges i altres informacions no estan operatives.
  • Control des d'una aplicació externa no funciona.
