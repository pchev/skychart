Instal·lació en Linux Debian o Ubuntu

El paquet per a Debian està disponible per poder-lo instal·lar automàticament usant apt-get.

Primer, esdeveniu rel o useu sudo en cada comandament.

Afegiu la signatura de Patrick Chevalley a APT: 

wget -q http://www.ap-i.net/pub/patrick_chevalley.gpg -O- | sudo apt-key add -

Editeu el fitxer /etc/apt/sources.list per afegir-hi la línia següent: 

deb http://www.ap-i.net/deb/ skychart/

Llavors actualitzeu la vostra llista de paquets aumb una línia de comandament: 

apt-get update

El programari pre-requisit normalment s'instal·la automàticament. Si no fos així, comproveu que disposeu dels paquets libgtk2.0-0, libglib2.0-0, libpango1.0-0, libjpeg62, libpng12-0, libsqlite3-0

També instal·leu Xplanet si voleu veure imatges dels planetes, i Indi per guiar el telescopi: 

apt-get install xplanet indi

i instal·leu el programari: 

apt-get install skychart

i en cas que disposeu d'espai sobrat: 

apt-get install skychart-pictures

Això és tot. Sortiu de la rel i teclegeu Skychart per engegar el programa.
