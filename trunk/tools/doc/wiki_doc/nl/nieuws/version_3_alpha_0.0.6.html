Versie 3 alpha 0.0.6

Deze versie voegt telescoop controle toe.

Een grote verbetering van deze versie bestaat uit de de telescoop controle via de INDI library voor zowel Linux als Windows. Ook kun je de plugins gebuiken van de vorige Windows CdC versie 2.7x.
Je kunt meer informatie vinden over het INDI project op indi.sourceforge.net.

De oculair en camera beeldvelmarkeringen zijn verbeterd. Je kunt nu ook een offset ten opzichte van het beeldcentrum instellen om het beeldveld van een auto-guider CCD of een off-axis volgsysteem af te beelden.
Deze markeringen kun je gebruiken om de telescoop positie te tonen, maar ook om te laten zien welke sterren je in de zoeker hebt.




Afbeeldingen van Planeten

In de afbeeldingen van de planeten kunnen nu ook reële oppervlak kenmerken getoond worden.

Deze optie is op dit moment alleen nog beschikbaar voor Windows en is niet standaard geactiveerd. Ga naar Instellingen → Zonnestelsel, het tab Planeten om deze instelling te activeren.

Andere veranderingen:

  • Opkomst / Ondergang tijsweergave
  • Knoppen om tussen de coordinaten rooster te schakelen
  • Knoppen om het aantal zichtbare objecten te veranderen
  • DDE server voor de Windows versie
  • Een commando toegevoegd dat compatibel met de vorige versie van DDE en TCP/IP
  • Het selecteren van de basis gemeenschappelijke map en de prive map voor de gebruikers data.
  • Veel bug correcties …
  • Waarschijnlijk nog veel ander dingen die ik vergeten ben…

De Linux binaries zijn nu beschikbaar als een gemakkelijk te installeren RPM.
