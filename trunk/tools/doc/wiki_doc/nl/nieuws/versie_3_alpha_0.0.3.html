Versie 3 alpha 0.0.3

Een nieuwe alpha versie is nu beschikbaar om tijdens de zomer mee te spelen.

Veel nieuwe mogelijkheden van versie 2.75 zijn toegevoegd.

Enkele echt nieuwe:

  • 2.8 million wereld steden database.
  • Galactische en Ecliptische kaart projectie.
  • Tijd aanpassing vanaf de tool bar.
  • Een coherent coordinaten rooster.
  • Rooster label naast de rand van de kaart.
  • Nieuw gevulde Melkweg weergave.

TCP/IP server mogelijkheid, kijk in de broncode pakketten voor meer informatie. Perl script voorbeelden zijn beschikbaar in de scripts/ map.
