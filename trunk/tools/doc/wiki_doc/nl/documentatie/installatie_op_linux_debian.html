Installatie op Debian of Ubuntu Linux

Het pakket voor Debian is via een automatische installatie verkrijgbaar doormiddel van apt-get.

Log in als root of gebruik steeds 'sudo' bij ieder commando.

Voeg Patrick Chevalley's GPG-signature toe aan APT: 

wget -q http://www.ap-i.net/pub/patrick_chevalley.gpg -O- | sudo apt-key add -

Voeg aan het bestand /etc/apt/sources.list de volgende regel toe: 

deb http://www.ap-i.net/deb/ skychart/

Ververs de lijst met pakketten met het commando: 

apt-get update

Installeer de essentiele software. Het maakt niet met welke X-server Debian werkt.

Je kunt de libraries uitbreiden door het toevoegen van de pakketten libgtk1.2, libgdk-pixbuf2, libjepg62 en libsqlite3-0 

apt-get install libgtk1.2 libgdk-pixbuf2 libjpeg62 libsqlite3-0

Installeer ook Xplanet om plaatjes van planeten af te beelden. 

apt-get install xplanet

Installeer vervolgens het programma: 

apt-get install skychart

en als je voldoende ruimte om je harddisk hebt: 

apt-get install skychart-pictures

Dat is alles, verlaat de root-shell en klik skychart om het programma te starten.

Opmerking: Bij de upgrade van een oude versie naar Versie 3 Beta 0.1.0 zal een directe automatische toevoeging niet gaan werken vanwege de veranderde dependencies. Installeer in dat geval de eerder aangegeven software opnieuw zoals hierboven is aangegeven.
