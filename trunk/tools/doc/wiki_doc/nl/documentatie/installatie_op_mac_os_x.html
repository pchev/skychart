Installatie op Mac OS X

Vereisten

  • Een recente Mac met Intel processor, PPC processors worden momenteel niet ondersteund.
  • Mac OS X 10.4 of recenter.

Instructies om de binaries te installeren op een Mac OS X.

  • Download het installatie bestand voor de Mac, bijvoorbeeld skychart-3.0.1.4-i386-macosx.dmg
  • Open het bestand om de virtuele disk te 'mounten'.
  • In de geopende disk bevind zich een map met de naam “Cartes du Ciel”.
  • Kopiëer deze map naar je Applicatie map.
  • Je kunt nu de virtuele disk umounten (sleep deze naar de prullenbak).

Start het programma

  • Open bij Applicaties - de Cartes du Ciel map.
  • Open het “Skychart” icoon.

Beperkingen

  • Telescoop bediening werkt niet.
  • Het downloaden van plaatjes en andere informatie werkt niet.
  • Controle door externe applicaties werkt niet.
