Gebruik van CDC V3 onder FreeBSD

Installatie en gebruik van CDC v3 alpha 0.0.6 voor Linux onder FreeBSD

Deze procedure moet nog aangepast worden voor de beta versie. Je hebt Kylix Qt niet meer nodig, maar je moet wel libgtk1.2 en libgdk-pixbuf2 installeren.

1 - Zorg dat je de 'Linux binary compatibility' geïnstalleerd en geactiveerd hebt. Je kunt het van ports krijgen, nog beter is het gebruik van sysinstall.

Gedetailleerde instructies kun je hier vinden:
http://www.freebsd.org/doc/en_US.ISO8859-1/books/handbook/linuxemu.html

2 - Download zowel “Kylix QT runtime RPM” en de “CDC v3 alpha 0.0.6 RPM” vanaf ap-i.net

www.ap-i.net/skychart/modules.php?op=modload&name=Downloads&file=index&req=viewdownload&cid=8

3 - Installeer de beide RPMs (als eerste Kylix). Start als root een shell en voer het commando uit: 

/compat/Linux/bin/rpm --install path/to/the/filename.rpm

4 - Installeer de graphics/linux-jpeg port. Als root: 

cd /usr/ports/graphics/linux-jpeg
make install clean

5 - Start het programma: /compat/linux/usr/bin/skychart

Tot nog toe lijkt alles goed te werken.

Voorlopig is dit een werkende oplossing, maar Linux emulatie voor Ciel is zeker niet wat FreeBSD gebruikers leuk zullen vinden. Het is gemakkelijker om naturel geporteerde compilaties van zowel Ciel als de Kylix bibliotheken te hebben.
FreePascal is beschikbaar voor FreeBSB, een naturel FreeBSD compilatie is nu mogelijk.

(Dit werkte voor mij met gebruik van 5.2.1 en linux_base-7.1_7)
