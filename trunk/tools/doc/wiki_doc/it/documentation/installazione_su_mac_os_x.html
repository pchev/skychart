Installazione su Mac OS X

Prerequisiti

  • Un sistema Mac recente dotato di processore Intel, i processori PPC non sono al momento supportati.
  • Mac OS X 10.4 o successivo.

Istruzione per l'installazione dei file binari su Mac OS X.

  • Scarica il file di installazione skychart-3.x.x.x-i386-macosx.dmg
  • Apri il file per montare l'hard disk virtuale.
  • L'hard disk virtuale contiene la cartella “Cartes du Ciel”.
  • Copia questa cartella nella tua cartella “Applicazioni”.
  • Smonta l'hard disk virtuale (muovilo nel cestino).

Lancia il programma

  • Apri la cartella Applicazioni - Cartes du Ciel.
  • Clicca sull'icona “skychart”.

Limitazioni

  • Alcune opzioni di configurazione non sono disponibili.
  • Il download di immagini e altre informazioni non funziona.
  • Il controllo da un applicazione esterna non funziona.
