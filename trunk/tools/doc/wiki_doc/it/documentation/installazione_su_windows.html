Installazione su Windows

Istruzioni per l'installazione dei file binari su sistemi Windows

E' possibile installare la versione 3.0 assieme alla versione completa 2.76 nella stessa cartella, oppure installare la sola versione 3.0.

Lancia il programma skychart_3.xxxxx.exe per compiere l'installazione nella directory di tua scelta. (es. C:\Programmi\Ciel)

Potrai poi utilizzare la versione 3 con il comando cdc.exe o la versione 2.76 con ciel.exe.
