Installation on Mac OS X

Prerequisites

  • A recent Mac with Intel processor, PPC processors are not supported at the moment.
  • Mac OS X 10.4 or better.

Instruction to install the binaries on Mac OS X.

  • Download the install file, i.e. skychart-3.0.1.4-i386-macosx.dmg
  • Open the file to mount the virtual disk.
  • The disk folder open, it contain a folder “Cartes du Ciel”.
  • Copy this folder to your Applications folder.
  • You can now umount the virtual disk (drag to the trash).

Start the program

  • Open the Applications - Cartes du Ciel folder.
  • Open the “skychart” icon.

Limitations

  • Telescope control does not work.
  • Download pictures and other information does not work.
  • Control by an external application does not work.
