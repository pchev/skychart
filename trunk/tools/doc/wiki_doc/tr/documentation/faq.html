Genel

CdC'ye nasıl katkıda bulunabilirim?

Birkaç şekilde katkıda bulunabilirsiniz:

  • Yazılımın son sürümünü deneyebilir ve bulduğunuz herhangi bir hatayı hata takipçisine kaydedebilirsiniz.
  • Metinlerdeki yazım ve dilbilgisi hatalarını ortaya çıkarmak için gönüllü olabilirsiniz.
  • Yazılımı veya bu sayfayı anadilinize çevirebilirsiniz.
  • Yazılımın kodunu gözden geçirebilir ve gerekli gördüğünüz iyileştirmeleri paylaşabilirsiniz.
  • Please send me your ideas, I am interested in your suggestions.

Kurulum

CdC 2.76 versiyonunu kullanıyorum. Yeni sürüm olan 3. versiyonu kullanmak için ne yapmalıyım? Ne kadar dosya yüklemem gerekli ve öncelikle 2.76'yı silmem şart mıdır?

3.0 beta sürümünü, 2.76 sürümünün olduğu klasöre yüklemeniz yeterli. Yeni sürümü farklı bir klasöre yüklemenize ya da eski dosyaları silmenize gerek yoktur. Bu şekilde her iki sürümü de bir arada kullanabilirsiniz. V3.0 tamamlanana kadar bu yararlı olacaktır. V3.0'ı çalıştırmak için cdc.exe'yi; V2.76'yı çalıştırmak için ise ciel.exe dosyasını kullabilirsiniz.

CdC V2.76 ile uyumlu olan önceki kataloglar CdC V3.0 ile de uyumlu mudur?

Evet, tüm bu kataloglar V3.0 ile kullanılabilir.

Where do I install the extra catalogues under Linux, in which directory do they go into?

If you can have root right it is best to put them in /usr/share/apps/skychart/cat to have them with the basic catalogs. But you can also install them anywhere as long as you specify the full path in the catalog configuration menu.

Can I get Cartes du Ciel software for a Macintosh

I recently compiled and ran CdC on OSX 10.4/Intel. This is probably also possible with the previous PPC versions. But this still need some work and I am not a Mac user, please contact me if you have some development skill and want to help with this port.

What can I do with MySQL

MySQL was used as the default database up to the version alpha 7. Since june 2005 it is replaced by the more simple SQLite. You can still use MySQL if required, for example to share the database across a local network. The database is used to store the following data: asteroids and comets elements, object pictures, world locations.

Kullanım

CDC V3.0'ın Linux versiyonu teleskop takibini destekliyor mu?

Evet. CdC V3.0 Linux versiyonu teleskopla bağlantı kurmak için INDI kullanır. INDI tarafından desteklenen herhangi bir teleskop modeli CdC ile sorunsuz çalışacaktır. Also you can use plugins to interface to ASCOM, Meade protocol or a simple encoder interface.

There is no “follow” and “see” telescope in the telescope menu of the V3.0

This is replaced by the “lock on” feature (the anchor button). When the telescope is connected and no other object is selected this button lock on the telescope.

Why is the sky blue

Because it is daytime or the Moon is up. You can change that from the configuration menu, Display, Sky color.

How can I see sky chart with simulation of planets before 3000 B.C

This is not possible because we lack a reliable computation method before 3000 BC. Even the JPL long ephemeride DE406 do not go before.

How can I switch the language from english to my language

Since the version beta 3.0.1.2 the language is detected automatically when you start the program. The menu Setup / System / Language let you select another language.

Does have CdC V3 DDE support

Not at the moment. The standard communication protocol for the V3 is tcp/ip for portability reason. DDE will probably be implemented in the future, maybe as a gateway.
