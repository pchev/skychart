Un nouveau catalogue idéal pour la cartographie

Le catalogue d'étoiles UCAC3 à été publié fin août 2009 par l'USNO.
Ces caractéristiques principales sont:

  • Couvre tout le ciel
  • 100 million d'étoiles
  • Complet jusqu'à la magnitude 16
  • Bonne résolution des étoiles double
  • Incroyablement complet dans les champs riches
  • Mouvement propre pour chaque étoile
  • Magnitude dans la bande 579-642 nm
  • Inclus la photométrie de 2MASS et de SuperCosmos
  • Indication de l'appartenance de l'étoile à d'autres catalogues
  • Les étoiles brillantes sont imprécises ou manquantes

Utilisation avec Cartes du Ciel

A partir de ces caractéristiques un catalogue à été construit pour l'utilisation avec Cartes du Ciel.
Les informations pour chaque étoile sont: identifiant, ascension droite, déclinaison, magnitude d'ouverture, mouvement propre, indice de couleur approximatif (venant de SuperCosmos B et R2).
Pour résoudre le problème des étoiles brillantes il n'inclut aucune étoile marquée comme présente dans Tycho2 et aucune étoile plus brillante que la magnitude 10. Cela implique que vous devez toujours activer le catalogue Tycho2 en même temps que ce catalogue.

Pour utiliser ce catalogue commencez par télécharger les parties que vous voulez depuis la page telechargement catalogue.
Il est découpé en trois parties pour que vous puissiez prendre uniquement les zones visible depuis chez vous.

  • ucac3-catalog-south.zip : zone sud -90 à -30
  • ucac3-catalog-equator.zip : zone équatoriale -30 à +22.5
  • ucac3-catalog-north.zip : zone nord +22.5 à +90

Un quatrième fichier ucac3-index.zip contient l'index de recherche pour toutes les zones.
Chaque fichier a une taille d'environ 500 MB.

Commencez par extraire tout les zip dans ciel\cat ou share/skychart/cat. Comme chaque fichier peut être utiliser individuellement il contient deux fichiers commun, quand on vous demande si il faut les remplacer vous pouvez répondre indifféremment oui ou non.

Ensuite depuis Cartes du Ciel ouvrez le menu Configuration -> Catalogues, cliquez sur Ajouter et sélectionnez le fichier 3uc.hdr. Cliquez dans la première colonne pour qu'elle deviennent verte.
Puis ouvrez l'onglet CdC étoiles pour activer Tycho2 et enlever le catalogue GSC. N'oubliez pas d'enregistrer les options.

Construisez votre propre catalogue

Si vous possédez une copie du DVD UCAC3 vous pouvez utiliser Catgen pour construire votre propre catalogue à partir des fichiers originaux. Cela vous permet d'inclure plus ou moins d'information pour chaque étoile selon vos besoins.

Vous devez d'abord convertir les fichiers binaires du DVD en des fichiers texte en utilisant le programme u3read inclut sur le DVD ou mon programme u3conv.
L'avantage de u3conv est de traiter tout les fichiers d'un coup et d'ajouter l'identifiant ainsi que l'indice de couleur après les autres informations. Il y a aussi une option pour inclure ou non les étoiles de Tycho2 et les étoiles brillantes.
Vous pouvez télécharger u3conv pour Windows et Linux ou le code source. Lancez le programme depuis une console pour afficher l'aide.

Utilisez ensuite Catgen pour convertir les fichiers texte dans un format binaire optimisé pour l'affichage de cartes. Vous pouvez commencer par charger le fichier du projet utilisé pour le catalogue standard.
Utilisez une version plus récente que skychart-3.1-1232 car j'ai corrigé un certain nombre de bug avec Catgen.
