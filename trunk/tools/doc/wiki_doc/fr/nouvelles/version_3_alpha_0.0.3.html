Version 3 alpha 0.0.3

Une nouvelle version alpha est disponible pour jouer pendant l'été.

Plusieurs fonctions de la version 2.75 ont été ajoutées.

Mais il y a aussi des vrais nouvelles fonctions:

  • Une base de donnée mondiale de 2.8 million de lieu.
  • Carte en projection Galactique et Ecliptique.
  • Incrément du temps dans la barre d'outils.
  • Amélioration des grilles de coordonnées.
  • Label des grilles près du bord de la carte.
  • Nouvelle représentation de la Voie Lactée.

Possibilité de serveur TCP/IP, regardez le code source pour plus d'information. Des exemples de script Perl sont disponible dans le répertoire scripts/.
