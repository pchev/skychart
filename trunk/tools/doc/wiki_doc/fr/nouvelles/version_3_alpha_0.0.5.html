Version 3 alpha 0.0.5

Base de donnée pour les comètes, labels, plus de fonctions d'impression.

Ajout d'une base de donnée pour les comètes
Selon le même principe que pour les astéroides., voir Version 3 alpha 0.0.4 pour plus de détails.

Si vous avez déjà créer la base de donnée des astéroides il faut utiliser a nouveau le boutton “Create Database” pour la mise a jour.

Un exemple de donnée (maintenant dans le répertoire data/planet) charge les comètes visible en se moment. Vous pouvez obtenir des éléments mis a jour depuis
ftp://cfa-ftp.harvard.edu/pub/MPCORB/COMET.DAT
ou
http://cfa-www.harvard.edu/iau/Ephemerides/Comets/SoftwareComets.html
dans ce dernier cas prenez bien le fichier au format MPC.
Avec Windows il faut faire attention de convertir le caractère de fin de ligne. Dans une prochaine version une fonction de téléchargement simplifiera cela.

Label des objets.
Selon le même principe que dans la version 2.75, avec l'utilisation de lettre grecs pour les étoiles brillantes.
Mais vous pouvez maintenant modifier la position ou le text de chaque label en utilisant un click droit.
Le click gauche sur n'importe quel label donne des informations détaillée sur l'objet. Les modifications de label sont enregistées avec les options par defaut ainsi que dans les cartes enregistrées.

Impression
Le menu de selection d'imprimante offre maintenant la possibilité d'imprimer directement ou de produire un fichier au format Postscript ou BMP.
Cela donne des images haute résolution des cartes, y compris avec le même rendu qu'a l'écran. Vous pouvez choisir une application pour traiter ou imprimer ces images.

Il y a une nouvelle option pour utiliser les coordonnées apparentes à la place des coordonnée moyenne. Cela applique les correction pour la nutation et l'aberation pour obtenir la position precise dans le référentiel de la Terre.

Beaucoup d'option par défaut ont changé, certaine pour corriger des bug ou imprecisions, aussi je recommande d'effacer (ou de placer en lieu sur) le fichier .cartesduciel.ini avant d'utiliser cette version.
