Version 3 beta 0.1.2

Traduction du programme et images en lignes

Le programme peut maintenant être traduit en plusieurs langues. Grâce au travail de traducteurs volontaires il est déjà disponible en anglais, catalan, italien, espagnol, ukrainien et français.

Si vous êtes intéressé pour le traduire dans une autre langue regardez http://www.ap-i.net/skychart/fr/traductions

Le système d'aide est modifié pour utiliser un extrait du wiki ( http://www.ap-i.net/skychart )
Il est minimal pour le moment mais vous pouvez contribuer à l'enrichir!




Images en lignes

C'est la même fonction disponible précédemment avec les CDrom Realsky mais tout ce dont vous avez besoin est une connexion internet.

Choisir la région du ciel qui vous interesse et utiliser le bouton “DSS” pour afficher l'image de la région correspondante.

Vous pouvez configurer la source de donnée utilisée depuis la configuration Images\DSS. Plusieurs sources intéressantes sont déjà configurée mais vous pouvez en ajouter avec le menu Internet\Images DSS.








Le générateur de catalogue CatGen est maintenant inclut dans le programme principal. Accédez cette fonction par le bouton “CatGen” dans la configuration des catalogues.
Vous pouvez toujours construire des catalogues binaires mais il est aussi possible d'utiliser directement des fichiers texte. Ces catalogues texte sont destiné a remplacer les catalogues externes de la version 2.7, l'avantage sur les catalogues binaires est qu'il n'y a pas besoin de reconstruire le catalogue quand les données sont modifiées. Un cas d'utilisation typique est une liste des dernières nova à afficher sur les cartes.

Le proxy internet peut être configuré dans Configuration/Internet/Proxy si nécessaire.

Des labels personnalisés peuvent être ajouté sur la carte par le menu du clic droit.

Des barres de défilement sur chaque carte permettent de naviguer rapidement sur tout le ciel. Attention qu'avec un fort niveau de zoom il ne faut utiliser que les flèches car le curseur est trop sensible.

Les labels des simulations peuvent être modifiés dans Configuration/Date-Heure/Simulation. Vous pouvez choisir combien de labels et le texte a afficher.

Beaucoup de corrections de bug grâce à tout vos tests. Le nouveau gestionaire de bug marche bien et sur 180 rapports 160 sont corrigés.
Environ 300 patchs ont été appliqué au code source pour cette version. Si le détail vous intéresse regardez http://skychart.svn.sourceforge.net/viewvc/skychart/trunk/?view=log
Il a pris plus de temps que d'habitude entre les versions 3.0.1.0 et 3.0.1.2. C'est parce que tout les tests ont pu être fait sur la version de développement 3.0.1.1 mise à jour chaque semaine pendant ce temps.
