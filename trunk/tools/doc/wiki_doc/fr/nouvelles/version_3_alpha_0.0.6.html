Version 3 alpha 0.0.6

Cette version ajoute le pilotage des télescopes.

La principale amélioration de cette version est le pilotage des télescopes utilisant la librarie INDI pour les version Linux et Windows, ou les mêmes plugins que la version 2.7 pour la version Windows.
Vous pouvez trouver plus d'information sur INDI a indi.sourceforge.net

L'affichage des champs d'oculaire et de caméra a été amélioré.
Vous pouvez maintenant indiquer un décalage du centre du champ, ce qui permet de dessiner le champ d'une caméra d'auto-guidage ou d'un oculaire off-axis.




Images de planètes
Les planètes sont maintenant représentée avec leurs détails de surface.

Cette option n'est disponible que pour la version Windows pour le moment et n'est pas active par défaut. Ouvrez le paramétrage Solar System / Planets pour l'activer.

Autres modifications:

  • Affichage des heures de lever et coucher.
  • Bouton pour afficher les grilles de coordonnées.
  • Bouton pour modifier le nombre d'objets visibles.
  • Serveur DDE server pour la version Windows.
  • Ajout de commandes compatibles avec la version précédente pour DDE et TCP/IP.
  • Choix du répertoire commun de base et d'un répertoire privé de l'utilisateur.
  • Beaucoup de correction de bug …
  • Et probablement d'autres choses dont je ne me souviens pas …

L'installation des binaires pour Linux peut maintenant se faire facilement a l'aide d'un fichier RPM.

Pour un maximum de confort installez aussi le rpm libborqt disponible ici.
