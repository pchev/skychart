Nouvelles pages web

Le site web de Cartes du Ciel est modifié pour aider aux contributions des utilisateurs

Le nouveau wiki vous permet de contribuer à ces pages, principalement pour la documentation du programme et les traductions.

Avec le snapshot du programme d'aujourd'hui un extrait de ce wiki est utilisé pour le système d'aide.

Pour le moment c'est plus un squelette qu'une vraie documentation. Les différents chapitres seront complétés au cours du temps.

Regardez les pages Contributions et Traductions à http://www.ap-i.net/skychart pour plus d'information.

Aucun changement est sans inconvénient, avec le remplacement de PostNuke par DokuWiki beaucoup de liens direct depuis des sites externes ne fonctionnent plus.
J'essaie d'au moins les rediriger sur la page principale mais vous êtes encouragé de mettre à jours vos liens si vous le pouvez.
