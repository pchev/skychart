Version 3 beta 0.1.6

Dernière beta avant la version stable

La prochaine version qui sera disponible plus tard cette année sera la première version stable de la V3.

Changement de l'emplacement d'enregistrement des fichiers de configuration

ATTENTION!
Cette version change l'emplacement d'enregistrement des fichiers de configuration.

Pour Windows c'est maintenant %LOCALAPPDATA%\Skychart et pour Linux ~/.skychart

La configuration du programme est réinitialisée aux valeurs par défaut.

Après la première exécution du programme vous pouvez recopier les fichiers que vous voulez conserver dans les nouveaux répertoires.

Ne PAS recopier cartesduciel.ini car il contient des chemins non valide.
Vous pouvez ensuite complètement effacer Mes Documents\Cartes du Ciel ou ~/cartes_du_ciel et ~/.cartesduciel.ini

Pour Linux le répertoire d'installation par défaut des données du programme est maintenant /usr/share/skychart, si vous aviez installé des catalogues supplémentaires déplacez les depuis /usr/share/apps/skychart.

Documentation:

La documentation du programme à pris un grand départ à l'initiative de Marceau GUIHARD.
Grâce à son travail et à celui de tout ceux qui collaborent au wiki il y a maintenant une documentation complète accessible directement depuis le programme.
Vous pouvez utiliser le bouton Aide ou la touche F1 n'importe ou dans le programme.

Horloge astronomique:

Une horloge astronomique est disponible dans le menu Voir/Horloge.

Le nouveau programme cdcicon montre cette horloge avec l'heure sidérale dans la barre de tâche.
Un clic droit montre un menu qui permet d'ouvrir directement le calendrier ou Cartes du Ciel.

L'installateur pour Windows propose de lancer ce programme automatiquement au démarrage.

Etoiles Variables:

Une nouvelle version de VarObs fait maintenant partie de Cartes du Ciel.

Cette version est modifiée pour les nouveaux format de fichiers de l'AAVSO, et les fichiers utilisateurs sont maintenant dans le même répertoire que Cartes du Ciel.

Autres changements:

  • Mise à jour des zones horaires pour Windows. Utilise les zone horaires du système sous Linux
  • La ligne du méridien utilise la couleur “autres lignes” pour plus de visibilité.
  • Affichage de la version du compilateur utilisé dans la fenêtre A Propos.
  • Il est maintenant possible de choisir la taille des étoiles en mode ligne pour l'affichage ou l'impression depuis le menu configuration/affichage.
  • Pour réduire le chevauchement des labels provenant de différentes sources sont écrit a différentes position autour de l'objet.
  • Ajout d'options de ligne de commande
  • Traduction en Islandais par Sverrir Gudmundsson.
  • Traduction en chinois par Lam Xiang.
