Commandes de la ligne de commande

CdC, accepte les options suivantes en ligne de commande :

Option Paramètres Fonction
–config chemin du fichier de configuration Permet de spécifier le fichier de configuration à utilisé a la place de ”%LOCALAPPDATA%\skychart\skychart.ini” ou ”~/.skychart/skychart.ini”
–unique Ne lance pas le programme si une autre instance est déjà active
–nosplash Ne montre pas l'écran de démarrage
–daemon Lance le programme en arrière plan sans montrer la fenêtre principale

Sous Unix vous pouvez envoyer un signal au programme avec la commande kill :

Signal Action
1, HUP Recharge la carte et les options par défaut
15, TERM Ferme le programme proprement
