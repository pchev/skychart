Installation pour Mac OS X

Prérequis

  • Un Mac récent avec un processeur Intel, les processeurs PPC ne sont pas supportés pour le moment.
  • Mac OS X 10.4 ou supérieur.

Instruction pour l'installation des binaires sur un système Mac OS X.

  • Téléchargez le fichier d'installation, par exemple skychart-3.0.1.4-i386-macosx.dmg
  • Ouvrez le fichier pour le monter.
  • Le disque monté s'ouvre, il contient un dossier “Cartes du Ciel”.
  • Copiez ce dossier dans le dossier Applications
  • Vous pouvez démonter le disque en le tirant dans la poubelle.

Lancement du programme

  • Ouvrez le dossier Applications - Cartes du Ciel
  • Ouvrez l'icone “skychart”

Limitations

  • Le pilotage des télescope ne marche pas.

* Les téléchargement d'images ou autres informations ne marche pas.

  • Le pilotage depuis une autre application ne marche pas.
