Calendrier, Crépuscule

Cet onglet donne l'heure des crépuscules du matin et du soir pour chaque date sélectionnée.
Le crépuscule nautique commence quand le soleil est 12° sous l'horizon. Les étoiles les plus brillantes sont visibles.
Le crépuscule astronomique commence quand le soleil est 18° sous l'horizon. Le ciel est noir et toutes les étoiles sont visibles.
