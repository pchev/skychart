Barre d'état

Côté gauche

Cette partie affiche en permanence les coordonnées du curseur sur la carte (coordonnées équatoriales apparentes).

Côté droit

Si un objet est sélectionné, cette partie affiche un extrait des informations détaillées sur l'objet :

  • Coordonnées équatoriales apparentes.
  • Première partie de la fiche Information détaillée (caracteristiques de l'objet, le type est codé comme dans la liste des objets).
