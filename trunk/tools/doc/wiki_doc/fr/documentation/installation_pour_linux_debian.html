Installation sur Linux Debian ou Ubuntu

Des paquet Debian sont disponibles pour une installation automatique avec apt-get

Se connecter en root ou utilisez sudo devant chaque commande.

Ajouter la signature de Patrick Chevalley à APT : 

wget -q http://www.ap-i.net/pub/patrick_chevalley.gpg -O- | sudo apt-key add -

Editer le fichier /etc/apt/sources.list pour ajouter la ligne suivante: 

deb http://www.ap-i.net/deb/ skychart/

Rafraichir la liste des paquets avec la commande: 

apt-get update

Les logiciel requis sont normalement installé automatiquement. En cas de problème vérifiez que vous avez les paquets libgtk2.0-0, libglib2.0-0, libpango1.0-0, libjpeg62, libpng12-0, libsqlite3-0

Installez aussi Xplanet pour afficher les images des planètes et Indi pour piloter le télescope. 

apt-get install xplanet indi

puis installez le programme: 

apt-get install skychart

et si vous avez beaucoup de place disque: 

apt-get install skychart-pictures

C'est tout, quitter le shell root et tapez skychart pour lancer le progarmme.
