Коментарі до версії 3.0.1.5 SVN

October 18 changes:

To reduce label overlap the label from diferent source are now draw at different position around an object. 
It is now possible to activate the labels for stars, variables, and double without overlap.
For the nebulae the SAC label is show on the right, label from other catalog is show on the left.
There is a new option in Setup/Display/Labels to show the labels from all stars catalog.
+ bug correction as show in roadmap: http://www.ap-i.net/mantis/roadmap_page.php

Зміни 21 вересня:

Додано ситуаційну допомогу (виклик натисканням F1).
Знову додано головне конфігураційне вікно.
Дороблений вибір порту телескопа в Linux.
Додано параметр командного рядка --config щоб конкретизувати конфігурацію .ini файлу.
Додано параметр командного рядка --unique щоб дозволити тільки один зразок для запуску.

Покращений діалог конфігурації.
Можна використовувати місцевий час для спостережень.
Робота із версією Skychart 3.0.

21 січня 2007
Зміни в налаштуванні часового поясу:
Починаючи із цієї версії, інформація про часовий пояс одержується із місця розташування обсерваторії, а не із комп`ютерних налаштувань. 
Головная перевага полягає у легкому керуванні правильним часом, якщо ви проводите обчислення для різних місць та для використання правильного DST для будь-якої дати в минулому або в майбутньому. Для цього вам потрібно встановити правильний часовий пояс в налаштуваннях “Обсерваторія”. Будьте впевнені у цьому, якщо у вашій країні більше, ніж один часовий пояс, та в будь-якому випадку, натиснуйте кнопку ОК для збереження налаштувань.

Інші зміни:
- Розширено формат дати в моделюванні.
- Мітка полюсу.
- Регулювання кольору неба для всіх систем координат.
- Кнопка для установлення часу 0h.
- Курсор мапи, як у версії 2.
