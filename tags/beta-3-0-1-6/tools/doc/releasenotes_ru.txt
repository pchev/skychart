Изменения в настройке часового пояса:
Начиная с этой версии, информация о часовом поясе берётся с места расположения обсерватории, а не из компьютерных настроек. 
Главное преимущество состоит в лёгком управлении правильным временем, если вы проводите вычисления для разных мест и для использования правильного DST для любой даты в прошлом и будущем. Для этого вам неоходимо установить правильный часовой пояс в настройках “Обсерватория”. Будьте в этом уверены, если в вашей стране больше, чем один часовой пояс, но в любом случая, нажимайте кнопку ОК для сохранения настроек.

Прочие изменения:
- Расширен формат даты в моделировании.
- Метка полюса.
- Регулировка цвета неба для всех систем координат.
- Кнопка для установки времени 0h.
- Курсор карты, как у версии 2.
