Export HTML Skychart wiki from a static copy of wiki at http://www.ap-i.net/static/skychart
It is comfigured with the include .htaccess who add do=export_html option.
This copy is updated daily by copy of the Dokuwiki data directory
The data directory is available at http://www.ap-i.net/static/skychart/data/pages/ and http://www.ap-i.net/static/skychart/data/media/
