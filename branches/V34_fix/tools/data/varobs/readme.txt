Variable Stars Observer
This directory contain the ephemeris input files by constellation 
for 16'000 stars extracted from the GCVS catalog.
http://www.sai.msu.su/groups/cluster/gcvs/gcvs/
http://cdsarc.u-strasbg.fr/viz-bin/Cat?II/250

Beware that the origin date for this ephemeris is far from now and 
the precision may suffer of that. 
Then this files must be considered as examples only.
